# Date

### When to use `Date` as a a field's `data type`
<details className='custom-accordion'>
  <summary>Date</summary>
  <p>Dates are very common to extract. Use this data type for dates. The dates can be be full so that it includes day, month and year,</p>
  <p>Dates are very common to extract. When you type a date in a `date` field, you must **mirror** the way the date is written on your document. I.e, don't format the date - type the date exactly as it is written on the document </p>
</details>

### How to annotate dates
When you annotate your training documents, you will want your input to **mirror** the document as closely as possible. `Date` is no exception from this rule. **Type what you see.**
