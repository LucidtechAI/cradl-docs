# Line item

## When to use a Line item field
  Line item is used for multiline layouts. The most common example of a line item is a list of products. It typically contains a product name, price, quantity, product code, etc.

A common mistake when creating a Line item field is to create a Line item field that corresponds to a singular line on your document. One Line item field can contain a whole table of items &lpar;which will save you *a lot* of time&rpar;.

## How to annotate a Line Item field
You do not need to annotate every line in a Line item field. It is sufficient to annotate a minimum of 3 lines if your Line item field is several lines long. 

So, if a table on your document has 20 lines, you can annotate 3 of the lines instead of all 20.
